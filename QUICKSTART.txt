   (Or use: pip3 install Flask)

   python app.py
   
   (Or use: python3 app.py)
   
   You'll see: "Running on http://127.0.0.1:5000"

❓ TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: "Module not found: Flask"
Solution: pip install Flask

Problem: "Port 5000 already in use"
Solution: Change port in app.py (last line):
         app.run(debug=True, port=5001)

Problem: "Database error"
Solution: Delete database/university.db and restart
         It will auto-create with fresh data

Problem: "Can't login"
Solution: Check you're using: student1/pass123 or faculty1/pass123
